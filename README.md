## 📖 Overview

MotoMarket is a **frontend web application for showcasing and browsing motorcycles online**.  
The website allows users to explore different motorbike models, view product information, and navigate through a clean and responsive interface.

This project focuses on **front-end development fundamentals**, including layout design, responsive UI, and interactive features implemented using vanilla JavaScript.

The project does **not use a backend or database**, making it a lightweight static website suitable for learning front-end development.

---

## ✨ Features

- 🏍️ Browse different motorcycle products  
- 🔍 View product details and images  
- 📱 Responsive design for multiple screen sizes  
- 🎨 Clean and modern user interface  
- ⚡ Interactive elements using JavaScript  
- 🌐 Fully static website (no backend)
